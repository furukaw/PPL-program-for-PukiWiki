# PPL-program-for-PukiWiki

「プログラミングおよびプログラミング言語ワークショップ」のプログラムを PukiWiki 記法にするスクリプト

### 使い方

Google Chrome で PPL の「プログラム」ページを開き、コンソールにスクリプト全てを貼り付けると、PukiWiki 記法のプログラムが返ります。

### 対象回
PPL 2018、PPL 2019 のプログラムで動作確認済みです。
